# Renderer_nj
real time renderer based on opengl
### Features:   
- OpenGL3.3 based
- PBR materials
- Post processing
- G-buffer
- skybox cubemap
- Kulla-Conty Approximation considering color
- Percentage Closer Filter
- Percentage Closer Soft Shadow
- Hi-Z Screen-Space Reflections
### Control:
- **WASD** moving positon
- **mouse** change yaw and pitch
- **scroll** change FOV
